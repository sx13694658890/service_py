# 知识库摘录（AI 问答）

本文档供 **知识库检索** 使用，与 [REQUIREMENTS.md](./REQUIREMENTS.md) 一致时可补充业务 FAQ。

## 产品是什么

首页提供 **AI 助手** 聊天窗口，帮助用户快速了解产品使用说明、常见问题。用户可通过文字提问或点击 **快捷问题** 发起对话。

## 如何使用

1. 点击页面上的 **聊天入口**（浮标）打开面板。  
2. 在输入框输入问题，或点击 **快捷问题**（将自动作为一条用户消息发送）。  
3. 等待助手回复；支持多轮追问。  
4. 回答依据 **知识库文档** 与模型能力生成，**仅供参考**，请以官方说明与人工客服为准。

## 人工客服

如需 **转人工**，可在快捷问题中选择「转人工」，或直接在对话中说明。人工服务时间以运营公告为准（示例：9:00–22:00）。

## 订阅与更新

若节点或套餐异常，可尝试 **更新订阅** 或 **重新导入** 配置后重试。若仍无法解决，请联系客服并提供账号与大致时间。

## 网页无法打开

请尝试更换浏览器、清除缓存、关闭代理或检查网络。若仅电脑端异常，可尝试移动端对比排查。
