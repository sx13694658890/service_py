# 故障排查与订单类问题

覆盖「用不了」、支付后未到账、订阅更新等常见咨询。

## 我用不了怎么办

请按顺序自查：

1. **网络**：确认设备联网，关闭可能影响访问的代理或 VPN 后重试。
2. **登录态**：若功能要求登录，请先登录再试；清除 Cookie 后需重新登录。
3. **额度与限流**：匿名用户可能有每日/每次限额；若提示超限，可稍后再试或登录后重试。
4. **订阅与节点**：若使用订阅类能力，确认当前账号套餐有效、节点或配置未过期；可尝试 **更新订阅** 或 **重新导入** 配置（见下文）。

若仍无法解决，请通过 **转人工** 或官方渠道联系客服，并尽量说明：账号（可脱敏）、大致时间、报错截图或文案。

## 付了钱没收到套餐

1. 先确认 **支付是否成功**（支付渠道订单状态）。
2. 等待 **系统同步延迟**（少数情况下需数分钟）；可刷新页面或重新登录后再看。
3. 核对 **登录账号** 是否与支付时使用的一致。
4. 若长时间未到账，请联系客服并提供：**订单号、支付时间、账号标识**（按客服要求提供）。

## 怎么更新订阅

若节点或套餐显示异常，可尝试：

1. 在客户端或页面中找到 **更新订阅** / **刷新** 类入口并执行。
2. 尝试 **重新导入** 订阅链接或配置（以产品实际界面为准）。
3. 仍异常时联系客服，并说明最近一次成功时间与当前现象。

以上内容仅供用户自助参考，具体按钮名称与流程以实际上线产品为准。
