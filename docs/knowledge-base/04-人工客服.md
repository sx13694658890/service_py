# 人工客服与转人工

## 如何转人工

用户可通过以下方式发起 **转人工** 诉求（具体入口以前端为准）：

- 在快捷问题中选择 **「转人工」**。
- 在输入框中直接输入包含 **「转人工」** 或 **「人工客服」** 等字样的消息。

系统将按产品策略返回 **转接说明**（例如服务时段、排队规则）；完整排队与工单系统可能为后续迭代能力。

## 服务时间

人工服务时间以 **页面公告或运营通知** 为准（示例：工作日 9:00–22:00）。非服务时段请留言、发送邮件或使用其他官方渠道，以实际产品提供为准。

## 与 AI 助手的分工

- **AI 助手**：7×24 可回答常见、文档内已覆盖的问题，速度快但 **不保证** 与最新政策完全一致。
- **人工客服**：处理账户、支付纠纷、复杂故障等需人工核实的问题。
