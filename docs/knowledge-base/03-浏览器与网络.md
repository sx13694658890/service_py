# 浏览器与网络

对应「网站在电脑打不开」等场景。

## 电脑端网站打不开

请依次尝试：

1. **更换浏览器**（如 Chrome、Edge 最新版本），排除浏览器插件或兼容性问题。
2. **清除缓存与 Cookie** 后重试，或使用 **无痕/隐私模式** 打开。
3. **关闭系统或浏览器代理**、第三方翻墙/广告拦截插件，再访问。
4. **检查本机网络**（能否打开其他网站；公司网络是否拦截域名）。
5. 若 **仅电脑异常、手机正常**，多为本机 DNS、代理或安全软件导致，可对比排查。

仍无法访问时，请联系网络管理员或客服，并提供：操作系统、浏览器版本、大致报错现象（空白页、证书错误、超时等）。

## 移动端对比

若手机可访问而电脑不可，重点检查电脑代理、Hosts、防火墙与企业网络策略。
